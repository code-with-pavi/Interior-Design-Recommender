# 📌 About The Project

AI Interior Designer is a smart web application developed using **Python** and **Flask** that analyzes uploaded room images and generates intelligent interior design suggestions based on:

- 🎨 Preferred Design Style
- 💰 Budget Range
- 🏡 Room Structure & Layout

This project combines Artificial Intelligence with modern web development to create a beginner-friendly yet professional AI application.

# 🧠 How It Works

### Step 1️⃣
User uploads a room image through the web interface.

### Step 2️⃣
Flask backend receives and stores the uploaded image securely.

### Step 3️⃣
The AI analyzer module processes the image and detects room characteristics.

### Step 4️⃣
The system generates intelligent interior design recommendations.

### Step 5️⃣
Results are displayed with personalized suggestions.
